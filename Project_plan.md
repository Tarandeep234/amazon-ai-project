# Project Title
AI-Powered Customer Review Intelligence and Product Recommendation System

# Business Problem
E-commerce platforms receive large volumes of customer reviews, but raw review text is difficult to use directly for marketing decisions.

# Project Objective
To analyze customer review sentiment and combine it with ratings and review activity to generate recommendation scores for product promotion and customer targeting.

# Tools
- Python
- VS Code
- SQL (Oracle)
- Azure
- Terminal (VS Code)

# Outputs
- Cleaned review dataset
- Sentiment-labeled reviews
- Product recommendation score table
- Charts and insights

# Version 1 Scope
- Load dataset
- Clean reviews
- Perform sentiment analysis
- Calculate product recommendation scores
- Export outputs
